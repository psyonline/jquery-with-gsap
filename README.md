# jquery-with-gsap
--

Extend jQuery to work with "GreenSock | TweenMax"(<http://greensock.com/tweenmax>).

#### Added Methods
- $()._animate(properties [, duration] [, easing] [, complete]);
- $()._stop([vars] [, target]);
- $()._css(properties [, value]);


*Add TweenMax's easings to jQuery.easing.
